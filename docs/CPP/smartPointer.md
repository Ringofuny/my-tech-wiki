# スマートポインタ
### cppのメモリ確保
- cppのメモリの動的確保ではnewやdeleteを使う
- この時newで確保したメモリをdeleteし忘れるとメモリリークが起きる
- こういう怖いことが起きないように**スマートポインタ**が生まれた

### スマートポインタ
- スマートポインタは動的確保の利用の際に生じるメモリの解放忘れ等の危険性を低減できる
- 確保したメモリをスマートポインタに渡すとそのメモリの所有権はスマートポインタの物になる。
- これはそのメモリの管理をスマートポインタにしてもらうようにするものであり、スマートポインタから使い、スマートポインタが削除するということである。スマートポインタを管理する人として考えるとわかりやすい。
