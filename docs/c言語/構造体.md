## 構造体について
構造体は仲間の項目（関連するもの）をひとまとめにして扱うことができるようにするもの

### 構造体のいいところ
1. [なんの変数か分かりやすくなる](###なんの変数か分かりやすくなる)

### なんの変数か分かりやすくなる
~~~c
typedef struct teachers {
	char name[50];
	int birth;
	int age;
} Data;

int main() {
	Data data;
	data.age = 17; // 年齢だなってすぐわかる
	printf("%d", data.age);
}
~~~
