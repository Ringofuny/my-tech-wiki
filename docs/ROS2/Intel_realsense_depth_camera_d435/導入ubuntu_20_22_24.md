## パッケージのインストール

1. サーバの公開鍵を登録
~~~bash
sudo mkdir -p /etc/apt/keyrings
curl -sSf https://librealsense.intel.com/Debian/librealsense.pgp | sudo tee /etc/apt/keyrings/librealsense.pgp > /dev/null
~~~

2. APT httpsサポートがインストールされているか確認
~~~bash
sudo apt-get install apt-transport-https
~~~

3. リポジトリのリストにサーバーを追加
~~~bash
echo "deb [signed-by=/etc/apt/keyrings/librealsense.pgp] https://librealsense.intel.com/Debian/apt-repo `lsb_release -cs` main" | \
sudo tee /etc/apt/sources.list.d/librealsense.list
sudo apt-get update
~~~

4. ライブラリのインストール
- 24.04ならdkmsは不要
~~~bash
sudo apt-get install librealsense2-dkms
sudo apt-get install librealsense2-utils
~~~

5. 開発者とデバッグパッケージをインストール
~~~bash
sudo apt-get install librealsense2-dev
sudo apt-get install librealsense2-dbg
~~~

## パッケージのアップデート

1. ローカルパッケージのキャッシュを更新
~~~bash
sudo apt-get update
~~~

2. Librealsenseを含むすべてのインストールされたパッケージをアップグレード
~~~bash
sudo apt-get upgrade
~~~

## realsense-rosのインストール

1. ワークスペースを作成
~~~bash
mkdir -p ~/realsense_ws/src
cd ~/realsense_ws/src/
~~~

2. realsense_rosをインストールしてcolcon build
~~~bash
git clone https://github.com/IntelRealSense/realsense-ros.git
cd ~/realsense_ws
colcon build --symlink-install
~~~

3. デフォルトブランチに
~~~bash
cd src/realsense-ros/
git branch
~~~

4. realsense_rosの実行
~~~bash
cd ~/realsense_ws/install
source ./local_setup.bash 
cd ..
ros2 launch realsense2_camera rs_launch.py
~~~
