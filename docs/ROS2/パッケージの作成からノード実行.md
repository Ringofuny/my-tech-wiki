## ノードとは
- ros2で実行される処理を担う最も基本的なオブジェクト
#### rclノード
- ROSノードの内部構造であり、基本機能の提供部分
- クライアントに依存しないＣ言語で書かれている

#### rmwノード
- rclノードの内部構造であり、DDSという通信価格を用いてROSノード間の通信を担う
- クライアントに依存しないＣ言語で書かれている

ノードについてはあり得ないほど詳しく解説しているサイトを見つけたのでこちらを参照

## 実際に書いてみる(hello world)
1. パッケージの作成
~~~bash
mkdir -p ~/ros_ws/src
cd ~/ros_ws/src
ros2 pkg create —-build-type ament_python pkg_name
~~~
2. これを実行するとpkg_nameというディレクトリができるのでそこへ移動
~~~bash
cd pkg_name
~~~
3. さらにその下へ移動
~~~bash
cd pkg_name
~~~
4. そこでPythonのコードを書く（Pythonの理由はパッケージを作るときにamdnt_pythonと指定したから）
~~~bash
nano source_name.py
~~~
5. Pythonコード
~~~Python
def main():
 print(“hello world”)
 
if __name__ == ‘__main__’:
 main() 
 
# 保存はctrl+o enter ctrl+x
~~~
6. 実行するためにsetup.pyを書き換える
~~~bash
cd ..
nano setup.py
~~~
~~~python
# 一部抜粋
‘consol_scripts’:[
    ‘pkg_name = pkg_name.source_name:main’
],
~~~
7. プロジェクトの直下に戻り設定を反映
~~~bash
cd ~/ros_ws
colcon build
source install/setup.bash
~~~
8. 実行
~~~bash
ros2 run pkg_name pkg_name
~~~
9. 結果
~~~txt
hello world
~~~
